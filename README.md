# Northline Digital

Premium UK digital agency site built with Next.js 15, TypeScript, and Tailwind CSS — including interactive `/work` case-study microsites.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — development server (Turbopack)
- `npm run build` — production build
- `npm run start` — serve production build
- `npm run lint` — ESLint

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP + ScrollTrigger
- Lenis smooth scroll
- Three.js (`@react-three/fiber`, `@react-three/drei`)
- Lucide icons
